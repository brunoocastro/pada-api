# Running locally

## Installation

```bash
# install project dependencies
$ npm install
```

## Configure DB Migration

```bash
# migrate dev db
$ npm run migrate:dev
```

## Start project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
